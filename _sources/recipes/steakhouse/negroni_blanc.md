# Negroni Blanc 

```{image} ../../images/
:alt: 
```

```{note}
* GLASS: Coupe
* GARNISH: Lemon Peel
```

```{list-table}
:header-rows:  1
* - Ingredients
  - Amount
* - Ford's Gin
  - 1-1/2 oz
* - St Germain
  - 1/2 oz
* - Littlet Blanc
  - 1 oz
```

## PROCEDURE
1. Build ingredients into a mixing glass with ice
2. Stir for 15-20 seconds
3. Strain into a coupe glass
4. Garnish with lemon peel
