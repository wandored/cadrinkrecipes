# Palm Beach Sling 

```{image} ../../images/
:alt: 
```

```{note}
* GLASS: Rocks
* GARNISH: Lemon Peel
```

```{list-table}
:header-rows:  1
* - Ingredients
  - Amount
* - Tito's Vodka
  - 1-1/2 oz
* - Giffard Pampelmousse Liquer
  - 1/2 oz
* - Lemon Juice
  - 3/4 oz
* - Grapefruit Juice
  - 1 oz
* - Simple Syrup
  - 1/2 oz
```

## PROCEDURE
1. Add ice.  Hard shake for 15 seconds
2. Strain into a rocks glass with ice
3. Garnish with lemon peel

```{important}
* All fruit juice needs to be Matalie's
*
```
