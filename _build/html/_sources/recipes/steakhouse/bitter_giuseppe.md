```{list-table}
:header-rows:  1
* - Ingredients
  - Amount
* - Cynar
  - 2 oz
* - Cocchi do Torino
  - 3/4 oz
* - Lemon Juice
  - 1/4 oz
* - Orange Bitters
  - 4 dashes
```
## PROCEDURE
1. Build in a shaker
2. Add ice, stir for 3 seconds
3. Strain into a rocks glass with a big cube
4. Garnish with lemon peel


```{tip}
* Great pre/post dinner drink
* Cynar is a versatile, herbaceous amaro made of artichokes.
```
